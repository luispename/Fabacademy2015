    "Double sided layer - PCB Milling Machine"

    After spending many hours thinking about my final project proposal I decided to build a machine that can do manufacture a provision of the double plate, as current machines have to finish one side of the plate, and need to replace the card in the another plate being careful not to change the position of the card. Once the plate is placed either to recalibrate the machine. I'm proposing to reduce or even eliminate the time spent in changing the face of the card use a camera to alignment of the double sided board

    A. Trouble

    There are many times I want to make a PCB for any electronics or mechatronics project and I have no space enougth using a typical PCB (only one layer side). The design is too large, or have a place to reduce PCB board have to use a double layer design.
    Once I have the full schematic design using any PCB software, proceed to build it dual path. Then I'll use a machine to manufacture the card. For that, the first step is milling one side, then the process stops and convert the card for milling or drilling the second design . I even have modern machines, this process is too basic, and spending too much time turning the card. Sometimes the holes do not coincide and we have a lot of problem with the PCB.

        The main idea behind this process is to ensure that when you turn your board around to mill the bottom side of your PCB it will be perfectly aligned with the top side.The mechanical alignment is accomplished by using alignment holes/pins. These are holes on your board and on the milling machine bed (typically a board of wood, known as “sacrificial” layer). Pins are used to align the holes on the board to the holes on the sacrificial layer. The holes are always pairs of holes that are symmetrical about a mirror axis. This ensures that when you turn your board around, the board can be aligned using the same alignment holes as before.

    C. First concept ideas

    To do this we will design and build a "Double sided layer - PCB Machine" This machine can perform drilling, milling and routing any card for maximum plate measures 10x10cm . Additionally have an automated system that allows for the case of plates with double layout, rotate the plate saving the time and effort that would cause change in a common PCB machine. It should be noted that this proposal PCB prototiping machine locates the same point and start milling and drilling holes for connecting plates with double layout, because otherwise, would have a full plate positioning failure, and the circuit would not be continuous on both sides of the plate. This prototype will have an automatic tool changer and an Exhaust fan

    D. Differential elements of the proposal

    A list of differential elements of the proposal

    1. Automatic Tool Changer
      • This prototype will have an automatic tool changer we want to make the machine will have the ability to fully automatic tool changes during the operation program, for example, it could use a milling tool and a drilling tool.
    2. Rotary axis
      • This machine will have an automated system that allows for the case of plates with double layout, rotate the plate saving the time and effort that would cause change in a common PCB machine.
    3. Image recognition
      • One of the tools that the project will have is a camera that recognize a couple of holes that were made in the plate. The camera will verify that the board is properly positioned , otherwise it will send a message to the computer indicating that it is poorly positioned . This camera produces us a reference in the X, Y axis which we can set the starting point to begin work. We will use the camera with image recognition, and compare this predetermined figure with a plate with holes. Once we compare it allows us to indicate whether similar or not, and according to that we can send a message or position the drill in the place of beginning.
    4. Exhaust fan
      • This machine will have an air extractor system that absorbs the copper particles removed from the plate. This system is activated whenever ending layout of a face, and can also be activated in the process of milling and drilling.
